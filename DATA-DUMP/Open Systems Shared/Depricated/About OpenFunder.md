# OpenFunder

## Democratized and Secure Crowdfunding.

Crowdfunding success rate is around 30-40%, with many of the failures
being exit scams.

Currently there is a large risk with crowdfunded campaigns to the user.
Many campaigns that show a lot of promise will not get funded, or
receive less funding to mitigate risk from the user.

OpenFunder solves these issues through democratization; the campaign
owner can create milestones or goals within the campaign (the more
detailed and granular the goals are, the more likely a user is to gain
confidence and fund the campaign). Each goal has promises or
requirements that must be met in order for a goal to be marked as
completed.

When the owner or developer finishes a goal, they will deliver and
present it. This is done on the OpenFunder platform. Users will review
the presentation, provide commentary and vote if it should pass. Only
once a goal passes, can the owner or developer receive funds to begin
work on the next goal.

If there are redflags with the project at any time, users can vote to
have all the funds returned. Users will have full control of what action
is taken with a project through voting.

An owner can request funds upfront for a goal, or only once it is
complete. The more reputation an owner has with previous projects, the
more likely they are to get more upfront funding.

A karma based system will be used for users, similar to Reddit. A user
cannot vote until they have the required karma. As a users karma grows,
so does the functionality of their account. This is to control the abuse
of fake accounts.
