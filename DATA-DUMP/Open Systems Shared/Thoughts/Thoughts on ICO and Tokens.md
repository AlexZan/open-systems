Beyond donating, investing will allow you to invest in a single work, a
project, an individual, or brand. Investing should open up funding far
beyond just that of donation.

1.  An artist can sell revenue share tokens in a single piece of art

2.  An artist can sell revenue share tokens for all their artwork, past
    and future.

3.  A tech hub group can sell revenue share tokens for all their past
    and future work, or just in a single tech.

4.  A user can sell tokens for a service or product they are creating,
    which users can redeem for that service or product.

#### Use Case: 3d printer

1.  User creates project to design, build and sell 3d printer

2.  Easily create a token, token exchange must be defined through smart
    contracts. These can be presets that can be selected when creating
    the project to make it simple for users to set common rules for
    their ICO.

    a.  If a token is redeemed, and the order is not processed within 7
        days, the original value at payment or the current value
        (whichever is greater) is refunded to the purchasers wallet.

    b.  If a claim is not addressed by the project owner within 7 days,
        the original value at payment or the current value (whichever is
        greater) is refunded to the purchasers wallet.

    c.  If a claim is voted as accepted, before the project owner
        proposes a resolution, and the claim resolution is not met
        within 7 days, the original value at payment or the current
        value (whichever is greater) is refunded to the purchasers
        wallet.

3.  The owner must then set a sale price for the token. This can have
    different values in different funding rounds or in the different
    goals of the project. E.g. Before the first goal of the project is
    funded, tokens will be sold at x value, before the second goal of
    the project is funded tokens will be sold at 1.2x, and so on.

4.  As soon as the project is posted, the ICO is live and users can
    begin purchasing tokens.
