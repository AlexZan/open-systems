Logic Network

A system which allows users to reuse logical components in debate,
creating a network of components which can provide statistical insight,
similar to the knowledge network, maybe it will be part of the knowledge
network.

Components:

## Claim

A claim is an attempt at a rational belief. A statement used to attempt
to describe reality.

## Argument

## Fallacy

A fallacy is a component that describes a common misconception.

## Bias

A cognitive bias is a component used to explain why a fallacy might be
occurring. It is a component that can be used as reasoning for a claim.
