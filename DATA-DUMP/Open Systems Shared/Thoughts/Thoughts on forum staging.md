Form staging is the thought that before a post makes it on the block
chain, it can be posted in a staging area which will only be on the
gossip network.

Here users can make suggestions about your post on how to improve it,
edits, etc.

There will be no experience flow in staging, meaning, users can\'t earn
experience for anything posted here, whether it\'s the initial post or a
response.

If a user breaks a rule here, such as tagging or certifying incorrectly,
users can let them know without using the reporting system. If the user
does not address these issues before they come out of staging, then they
can be reported.

## When a post goes out of staging, should users gain experience for their responses? 

Yes, this could be the incentive to provide help in the staging area.
Soon as the post is out of staging, users that subscribed to the post
can get notified, and voting can take place.

## Should a post require a consensus group before it goes out of staging?

Maybe once a day has passed in staging, if there are no issues that are
indicated through responses. The user can request to post out of
staging. At this point a consensus group will form using the subjects
the post has been tagged with. If everyone in the consensus group
acknowledges the post and does not block it, it will be posted to the
block chain. The request to post it to publish the post will cost the
funds to post it to the blockchain, if it is denied, the difference is
refunded.
