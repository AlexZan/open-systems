What should be on blockchain, smart contract and what should be on p2p?

Blockchain is a decentralized ledger that keeps an immutable history of
data changes

Smartcontract is decentralized code that can execute changes to a
blockchain

P2p is a decentralized data store and code, and can be immutable.
