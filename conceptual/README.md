# Conceptual - Organized Theory Layer

**Status:** Living, actively maintained
**Layer:** 1 (Conceptual - "What & Why")
**Source of Truth:** Yes (for philosophical foundations)

## Purpose

This directory contains **synthesized, organized concepts** extracted from the historical archive and refined for clarity and consistency.

## Relationship to Archive

- **Archive** = Raw historical notes (frozen snapshot)
- **Conceptual** = Curated synthesis (living documents)

Each concept document:
- Links back to `archive/` sources for provenance
- Resolves contradictions found in historical notes
- Represents **current best thinking**

## Document Status

| Status | Meaning |
|--------|---------|
| Draft | Work in progress, open to major changes |
| Review | Stable, seeking feedback |
| Accepted | Canonical, only minor edits |
| Deprecated | Superseded by newer concept |

## Adding New Ideas

### Rough/Unformed Ideas
Create a file in `conceptual/drafts/idea-YYYY-MM-DD.md`

### Formalized Concepts
Create `concept_[topic_name].md` following the template in `_template.md`
